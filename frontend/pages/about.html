Présentation du Projet

Cette interface de chat IA a été développée dans le cadre d'un projet scolaire afin de mettre en pratique des compétences en développement web, notamment l'utilisation de JavaScript Vanilla, du HTML sémantique et du CSS accompagné de bibliothèques CSS. Je me suis également lancé le défi d'apprendre et d'utiliser les Web Components.

Technologies Utilisées

Frontend

  • • HTML Sémantique
  • • Tailwind CSS
  • • JavaScript
  • • Web Components

Backend

  • • Python
  • • Flask
  • • API Ollama
  • • JSON (pour le stockage et la configuration)
